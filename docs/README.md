# مستندات پروژه memostyles

این پوشه مبنای برنامه‌ریزی، طراحی و توسعه فازبه‌فاز فروشگاه اینترنتی `memostyles` است.

هدف پروژه ساخت یک فروشگاه تخصصی اکسسوری زنانه با دو سطح اصلی است:

- فروشگاه مشتریان با تمرکز روی تجربه خرید، SEO، محصول، سبد خرید، پرداخت و پیگیری سفارش.
- پنل مدیریت فروشگاه با تمرکز روی محصول، موجودی، سفارش، مشتری، تخفیف، محتوا و گزارش‌ها.

## تصمیم اصلی پروژه

مسیر توسعه پیشنهادی:

```text
Backend-first + ERD-first + API-contract-first
```

یعنی ابتدا دامنه کسب‌وکار، دیتابیس، قوانین سفارش، موجودی و پرداخت را محکم می‌کنیم؛ سپس فروشگاه و پنل مدیریت را روی API پایدار می‌سازیم.
